# Exercise 1: Advanced Repository Management

Build a complete Git workflow for a team project with GitFlow.

## Objectives

- Set up advanced repository structure
- Implement GitFlow workflow
- Practice advanced merge strategies
- Create release branches and tags

## Tasks

### Task 1.1: Set up advanced repository structure
- Initialize Git repository
- Configure Git with best practices
- Create comprehensive .gitignore

### Task 1.2: Implement GitFlow
- Create main and develop branches
- Create feature branch with proper naming
- Use conventional commit messages

### Task 1.3: Advanced merge strategies
- Merge feature to develop
- Create release branch
- Merge release to main and tag
- Merge release back to develop

## Usage

```bash
cd 07-Exercise-1-Repository-Management
bash setup-repository.sh
```

## Expected Results

After running the script, you should have:
- A Git repository with GitFlow structure
- Main, develop, and feature branches
- Release branch merged to main with tag
- Clean commit history with proper messages

## Learning Outcomes

- Understand GitFlow workflow
- Practice merge strategies
- Create and manage release branches
- Use conventional commit messages
