# Exercise 2: Conflict Resolution Mastery

Master complex merge conflict scenarios and resolution strategies.

## Objectives

- Create complex conflict scenarios
- Resolve three-way merges
- Practice different conflict resolution strategies
- Understand merge conflict markers

## Tasks

### Task 2.1: Create complex conflict scenario
- Set up initial configuration file
- Create multiple branches with conflicting changes

### Task 2.2: Create multiple conflicting branches
- Database upgrade branch
- Security hardening branch
- Performance tuning branch

### Task 2.3: Complex three-way merge resolution
- Merge branches sequentially
- Resolve conflicts manually
- Combine changes from multiple branches

### Task 2.4: Advanced conflict resolution techniques
- Test different merge strategies
- Use ours/theirs strategies
- Prefer specific versions in conflicts

## Usage

```bash
cd 08-Exercise-2-Conflict-Resolution
bash setup-conflicts.sh
```

## Expected Results

After running the script, you should have:
- Multiple branches with conflicting changes
- Resolved merge conflicts
- Understanding of different resolution strategies

## Conflict Resolution Strategies

1. **Manual Resolution**: Edit files to combine changes
2. **Ours Strategy**: Keep our version completely
3. **Theirs Strategy**: Keep their version completely
4. **Prefer Ours**: Use our version when conflicts occur
5. **Prefer Theirs**: Use their version when conflicts occur

- Resolve complex merge conflicts
- Understand conflict markers
