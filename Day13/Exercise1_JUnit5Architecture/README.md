# Exercise 1: JUnit 5 Architecture & Fundamentals

## Mô tả
Bài tập này tập trung vào kiến thức cơ bản về JUnit 5 Architecture, Test Lifecycle, và Advanced Assertions.

## Nội dung
- CalculatorTest: Demo test lifecycle với @BeforeAll, @BeforeEach, @AfterEach, @AfterAll
- TestInfoInjectionTest: Demo test information injection và repeated tests
- AdvancedAssertionsTest: Demo các loại assertions nâng cao
- UserAssertions: Custom assertion class cho User domain object

## Chạy tests
```bash
mvn test -Dtest=com.xink.training.day13.exercise1.*
```
