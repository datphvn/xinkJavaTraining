# Exercise 2: Parameterized Testing

## Mô tả
Bài tập này demo các kỹ thuật Parameterized Testing với nhiều nguồn dữ liệu khác nhau.

## Nội dung
- ParameterizedTestingTest: Demo @CsvSource, @ValueSource, @EnumSource, @CsvFileSource, @MethodSource, @ArgumentsSource
- AdvancedParameterizedTest: Demo các pattern nâng cao với custom delimiters và multiple parameter sources

## Chạy tests
```bash
mvn test -Dtest=com.xink.training.day13.exercise2.*
```
